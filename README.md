## 功能
目前具有下面这些功能：
- 内置 `cd`，`exit` 等命令
- 具有可编程的 `if - then - else - fi` 功能
- 信号处理
- 内置环境变量
- 类似 `bash` 标准提示符
- 可以使用 shell 变量
- 具有命令行解析功能
- 功能模块化设计
- 后续更新

## 编译运行
```
make

./smsh
```
